Create CSR format sparse matrix from dense matrix, possibly thresholding.

Caller responsible for freeing Or, Ic, sA when done.

Parameters
[in]mnumber of rows.
[in]nnumber of cols.
[in]dAelements of dense matrix.
[in]dBelements of second dense matrix (can be null).
[in]epsabsolute threshold to consider element non-zero (can be 0.0 for no . thresholding).
[out]nnznumber of non-zeros.
[out]Orrow offsets.
[out]Iccolumn indices.
[out]sAnon-zeros elements of dA.
[out]sBnon-zeros elements of dB. (is dB is null, this is unused).
Returns
SD_OK if conversion succeeds, SD_ERR if error occurred.
