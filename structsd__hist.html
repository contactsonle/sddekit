void(* sd_hist::get) (sd_hist *h, double t, double *aff)

Get delayed data from history buffer.

If a and e denote afferent and efferent coupling terms, then this compute a[i] = e[vi[i]](t - vd[i]).

Note
len(aff) == nd
Parameters
[in,out]hhistory instance; if NULL, call is no-op.
[in]tcurrent time.
[out]affvector of afferent coupling terms, determined by vi/vd.
